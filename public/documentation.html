My Flix

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

REQUEST URL HTTP METHOD RESPONSE EXAMPLES
Get a list of movies /movies GET Returns a JSON list of movies. localhost:8080/movies
Get a list of users /users GET Returns a JSON list of users. localhost:8080/users
Get data about a movie from title /movies/:Title GET Returns a JSON data about a specific movie. localhost:8080/movies/Blade
Get data about user /users/:Username GET Returns a JSON data about a user localhost:8080/users/leeno
Get data about a genre /genres/:genre GET Returns a JSON data about a genre localhost:8080/genres/Action
Get data about a director /director/:directorName GET Returns a JSON data about director. localhost:8080/director/Lana%20Wachowski
Add new user /users POST Will add new user to JSON object. localhost:8080/users
Example inputs:
UserName: xxxxx,
Password: xxxxxx,
Email: xxxxxx,
Birthday: xx.xx.xxxx
Add new movie to list of favorite /users/:Username/movies/:MovieID POST Returns a text confirming which movie was added successfully. localhost:8080/users/leeno/movies/61e44b662fe00c3d6684946e
Allow users to update their user information /users/:Username PUT Returns the data fields that was updated by the user. localhost:8080/users/leeno
Example inputs:
Password: Newpassword
Delete the user /users/:Username DELETE Return a text confirming that the user was deleted. localhost:8080/users/leeno
Delete movie from list of favorite /users/:Username/movies/:MovieID DELETE Returns a text confirming which movie was removed successfully. localhost:8080/users/leeno/movies/61e44b662fe00c3d6684946e
